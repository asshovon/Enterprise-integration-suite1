# Enterprise Integration Suite - Live Local Execution

 সম্পূর্ণ অফলাইন এবং লোকাল মোডে তৈরি করা হয়েছে। 

আপনি যদি **Node.js** বা কোনো টার্মিনাল কমান্ড ব্যবহার করতে না চান, তবে নিচের যেকোনো একটি নিয়মে খুব সহজেই **Google Chrome** ব্রাউজারে এটি সরাসরি ওপেন করতে পারবেন।

---

## 🛠️ অপশন ১: এক-ক্লিক ডাবল-ক্লিক রানার (Python এর মাধ্যমে - সবচেয়ে সহজ)

আপনার কম্পিউটারে **Python** ইন্সটল করা থাকলে (যেকোনো আধুনিক কম্পিউটারে স্বয়ংক্রিয়ভাবে থাকে), আপনি কোনো কমান্ড টাইপ না করেই ডাবল ক্লিক করে প্রজেক্টটি চালু করতে পারবেন।

### উইন্ডোজ (Windows) কম্পিউটার ব্যবহারকারীদের জন্য:
১. এই ফোল্ডারের ভেতর থাকা **`run_local.bat`** ফাইলটি ডাবল ক্লিক করে রান করুন।
২. এটি স্বয়ংক্রিয়ভাবে ব্রাউজার ওপেন করে প্রজেক্টটি স্ক্রিনে চালু করে দেবে।

### ম্যাক/লিনাক্স (Mac/Linux) ব্যবহারকারীদের জন্য:
১. **`run_local.sh`** স্ক্রিপ্টটি ডাবল ক্লিক করুন অথবা টার্মিনালে `./run_local.sh` লিখে রান করুন।

## 🛠️ অপশন ২: VS Code "Live Server" এক্সটেনশন (Visual Code দিয়ে)

আপনি যদি কোড এডিটর ব্যবহার করেন, তবে কোনো সফটওয়্যার ইন্সটল করা ছাড়াই সরাসরি ব্রাউজারে দেখতে পাবেন:

১. **Visual Studio Code** এ এই প্রজেক্ট ফোল্ডারটি ওপেন করুন।
২. VS Code এর বাম পাশের **Extensions** ট্যাব থেকে **`Live Server`** (Ritwick Dey এর তৈরি) এক্সটেনশনটি সার্চ করে **Install** করুন।
৩. নিচে ডান কোনায় থাকা **`Go Live`** বাটনে ক্লিক করুন।
৪. আপনার **Google Chrome** ব্রাউজারে প্রজেক্টটি ৩০০০ বা ৫৫০০ পোর্টে সুন্দরভাবে রান হয়ে যাবে।

---

## 🛠️ অপশন ৩: পাইথন টার্মিনাল কমান্ড (সহজ ম্যানুয়াল উপায়)

আপনার সিস্টেমে (কম্পিউটারে) ইনস্টল থাকা পাইথনের সাহায্যে ১ সেকেন্ডে লোকাল সার্ভার তৈরি করুন:

### উইন্ডোজ / ম্যাক বা লিনাক্স:
১. এই ফোল্ডারের ভেতর কমান্ড প্রম্পট বা টার্মিনাল চালু করুন।
২. নিচের যেকোনো একটি কমান্ড ব্যবহার করুন:

**উইন্ডোজের জন্য:**
```cmd
python -m http.server 3000
```

**ম্যাক বা লিনাক্সের জন্য:**
```bash
python3 -m http.server 3000
```
৩. এবার আপনার **Google Chrome** ব্রাউজারে এড্রেসবারে লিখুন: 👉 **`http://localhost:3000`**

## 💻 C# .NET Core integration
এই প্রজেক্টে `.dotnet-backend` ফোল্ডারের ভেতরে সম্পূর্ণ এন্টারপ্রাইজ সিস্টেমের C# ASP.NET Core ক্লাসের ফাইলগুলো যুক্ত রয়েছে:

- `Controllers/IntegrationController.cs` (অ্যাকশন কন্ট্রোলার)
- `Models/EnterpriseModels.cs` (ডেটা স্কিমা ও রিকুয়েস্ট মডেল)
- `Program.cs` & `appsettings.json` (সিস্টেম বুট ও এন্টারপ্রাইজ কনডিউশন সেটিংস)

যা আপনি যেকোনো সময়ে রিয়েল .NET কোর প্রোডাকশন ব্যাকএন্ডে সহজে ইন্টিগ্রেট করে ব্রাউজার এবং রিয়েল সিস্টেমের কানেকশন তৈরি করতে পারবেন।
